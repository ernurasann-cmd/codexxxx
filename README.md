# Career Guidance Platform

Полнофункциональный backend для платформы профориентационных тестов, описанной в техническом задании. Реализован на FastAPI и использует SQLAlchemy для доступа к базе данных, поддерживает разграничение ролей (админ, профориентолог, клиент), управление тестами, назначениями и подписками, а также генерацию PDF-отчётов.

## Возможности

- Регистрация первого администратора и полноценная система авторизации по JWT.
- Управление пользователями администраторами (создание профориентологов и клиентов).
- Управление тарифами и выдача подписок профориентологам.
- Создание и публикация тестов с вопросами и вариантами ответов.
- Личный кабинет профориентолога: ведение клиентов, назначение тестов, формирование заключений и скачивание PDF.
- Личный кабинет клиента: прохождение тестов, просмотр результатов, скачивание PDF.
- Общедоступные сведения о тестах и выдача индивидуальных ссылок по токену.

## Стек

- Python 3.11+
- FastAPI
- SQLAlchemy (SQLite по умолчанию, легко переключить на PostgreSQL/MySQL через переменную окружения `DATABASE_URL`).
- JWT (python-jose) и bcrypt для хеширования паролей.
- ReportLab для формирования PDF-файлов.


## Запуск

1. Установите зависимости:

   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. (Опционально) Создайте файл `.env` и задайте переменные:

   ```env
   SECRET_KEY=смените-на-секрет
   DATABASE_URL=postgresql+psycopg2://user:password@localhost:5432/career
   ALLOW_ORIGINS=["http://localhost:3000"]
   ```

3. Запустите сервер:

   ```bash
   uvicorn app.main:app --reload
   ```

База данных создаётся автоматически при старте.

## Основные эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| POST  | `/auth/register` | Регистрация первого администратора |
| POST  | `/auth/token` | Получение JWT |
| GET   | `/users/me` | Данные текущего пользователя |
| POST  | `/admin/users` | Создание пользователей (админ) |
| POST  | `/admin/assessments` | Создание теста |
| POST  | `/professional/clients` | Создание клиента профориентологом |
| POST  | `/professional/assignments` | Назначение теста клиенту |
| POST  | `/client/assignments/{id}/submit` | Сдача теста клиентом |
| GET   | `/professional/assignments/{id}/pdf` | Скачивание отчёта профориентологом |
| GET   | `/client/assignments/{id}/pdf` | Скачивание отчёта клиентом |

Полное описание запросов и ответов доступно в интерактивной документации FastAPI по адресу `http://localhost:8000/docs`.

## Дополнительно

- В логике подписок реализована проверка на активный тариф при назначении тестов.
- Заложена возможность архивации завершённых назначений и восстановления из архива.
- Генератор PDF формирует базовый отчёт, который легко расширить.
